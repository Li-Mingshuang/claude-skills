context-injector
自动化 工具类
每次发消息时自动注入上下文——时间、天气、Git 分支状态等。Claude 不用你说就知道"现在几点、你在哪个分支"。通过 UserPromptSubmit hook 实现，配置文件控制开关，支持自定义数据源。
# 安装
git clone https://github.com/Li-Mingshuang/claude-context-injector
cp -r claude-context-injector/skill ~/.claude/skills/context-injector
📖 课程页 ⭐ GitHub
📓
记录（notes-diary）
知识管理 Obsidian
一句话记日记或笔记。说"记日记"写入 Journal/ 带时间戳和 AI 精炼反馈，说"记笔记"创建格式化 Markdown 并自动更新索引。30 分钟内的感悟合并到同一时间块，意图模糊时主动确认。
# 安装
git clone https://github.com/Li-Mingshuang/claude-skill-notes-diary
cp -r claude-skill-notes-diary ~/.claude/skills/记录
📖 课程页 ⭐ GitHub

🚀 通用安装方式

1
确认 Claude Code 已安装，skills 目录在 ~/.claude/skills/
2
Clone 对应仓库，将 skill 目录复制到 ~/.claude/skills/<skill-name>/
3
重启 Claude Code，在对话中说出触发词即可使用
4
部分 skill（如 context-injector）有 hook 配置，首次运行时 skill 会自动完成初始化
